Detailed Description

Глобальный контроллер для добавления общих атрибутов модели во все контроллеры.

Используется для внедрения роли пользователя (userRole) во все модели контроллеров, чтобы можно было использовать её в шаблонах представлений.

String org.example.controllers.GlobalControllerAdvice.userRole ( Authentication authentication)

Добавляет роль текущего пользователя в модель.

Метод вызывается автоматически перед выполнением каждого контроллера и добавляет атрибут "userRole" в модель, основываясь на данных аутентификации.

Parameters
authenticationОбъект аутентификации, содержащий данные о текущем пользователе
Returns
строка, представляющая роль пользователя (например, "ADMIN", "GUEST")
