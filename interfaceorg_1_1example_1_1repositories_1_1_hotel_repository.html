Detailed Description

Репозиторий для управления сущностями Hotel (Отели).

Содержит методы поиска отелей по различным критериям, включая город, доступность номеров и управляющего пользователя.

Найти отели с доступными номерами на заданные даты и по заданному количеству взрослых.

Отели подбираются с учетом:

  • города,
  • вместимости номеров,
  • отсутствия пересечения дат с уже существующими бронями.
