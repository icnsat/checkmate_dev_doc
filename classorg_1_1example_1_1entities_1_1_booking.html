Detailed Description

Сущность, представляющая бронирование номера в отеле.

Содержит информацию о бронировании, включая данные пользователя, информацию о номере, даты проживания и статус бронирования.

@entity Аннотация JPA для сопоставления с таблицей "Bookings" @table Указывает имя таблицы в базе данных
